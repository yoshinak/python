# html_cache

A simple html cache.

## Getting Started

Download this directory.
Sample to use is tests/html_cache_test.py#test_get_html_cache_not_exists().

### Prerequisites

Tested Python 2.7.6.

## Running the tests

---
python -m unittest discover -s html_cache/tests -p "*_test.py"
---
